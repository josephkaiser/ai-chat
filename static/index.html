About
Why this app exists
Joseph Kaiser

Thank you for trying AI Chat. The goal is a focused local coding assistant that stays useful without piling on unnecessary product layers.

The project is meant to be enjoyable to use, honest about what it is, respectful of the user, and economical with hardware while still being genuinely useful.

If you run into bugs or have ideas to improve it, please post an issue on GitHub. Your attention and contributions are incredibly valuable.
