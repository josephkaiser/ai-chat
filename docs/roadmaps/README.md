# Roadmaps

This folder holds longer-term planning artifacts that are useful for contributors but should not live in the repo root.

- `file-session-first-draft-runtime.md`: product/runtime direction for the file-session-first document workflow
- `llm-harness-core-roadmap.json`: structured capability roadmap for the core agent and runtime direction

Runtime-generated `.ai/` artifacts still exist inside workspaces when the app is running, but those are local state and are intentionally not tracked in the repository.
